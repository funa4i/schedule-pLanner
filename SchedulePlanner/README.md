# Запуск приложения

Создать миграцию для своей базы данных :

Запускать из корневой папки проекта

```bash
dotnet ef migrations add InitialMigration
```

### Через Docker

1. Перейти в корневую папку проекта.
2. Выполнить команду:

```bash
docker-compose up -d
```

3. API будет доступно по адресу:

```
http://localhost:8081
```

### Запуск без Docker

1. Открыть `appsettings.json` и изменить параметры подключения к БД на свои.

2. Запустить проект через Visual Studio или командой:

```bash
dotnet run
```

4. API будет доступно на указанном в настройках порту 5191.
